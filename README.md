# Projects
About the Cloud Classification project, I used SWIMCAT dataset for classifying 6 different classes of clouds.
Since the dataset is small, I made different generations of the dataset, so I could expand it.
Later I needed to use CNN architecture of the model, since I needed to use image classification.
The architecture contains Dense and GlobalAveragePooling2D layers.
After making architecture I needed to experiment with the number of epochs to train the model and fine-tuning it.

Next project is Doppler effect, where I needed to predict the speed of sound which is coming closer to the listener.
I used physics formula for Doppler effect and Discrete Fourier transformation.

MinesWeeper project uses basics of Python language, such as for and if loops.
